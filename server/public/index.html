What is Mangasaki?

Mangasaki is an innovative social network for manga and anime lovers. Create collections, scan books with AI, recommend titles to your friends, customize your profile, track what you're reading, have finished, and much more.


Mangasaki is not a manga readerâ€”it's a vibrant community where users can interact, discover hidden gems, and share their passion for manga and anime. Built with AI-powered scanning features and community-driven recommendations, Mangasaki empowers fans to:

  • Organize personalized manga collections.
  • Discover popular titles through user suggestions.
  • Track reading progress with ease.
  • Connect with friends and fellow fans in a shared space.

Development Environment

Mangasaki is an open-source project developed with modern tools:

  • Backend: Node.js with WebSockets
  • Frontend: Flutter (for mobile platforms)
  • Version control: Git + GitHub

You can explore the code on GitHub:
